Let’s break this:

🔸 ...formData

Copies old data (spread operator)

🔸 [e.target.name]

Gets input name dynamically
Example:

If typing in name field → name

If typing in email field → email

🔸 e.target.value

Gets the typed value

So basically:

=> Update only the field that user is typing in
=> Keep other fields unchanged
